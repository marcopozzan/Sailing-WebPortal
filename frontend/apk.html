📱 App Tablet - Download

App Android per la registrazione dati di bordo (GPS, vento, manovre). Da installare sui tablet di squadra prima della regata.

📖 Come installare sul tablet

  1. Scarica il file .apk qui sotto sul tablet
  2. Apri il file scaricato (di solito da Download del browser)
  3. Android chiederà il permesso di "installare app sconosciute" per il browser/file manager con cui hai aperto il file → tocca "Impostazioni" e abilita il toggle
  4. Torna indietro e completa l'installazione
  5. Apri "SOAR" dal menu app per iniziare

Le versioni successive si aggiornano dalla stessa pagina senza dover disinstallare la precedente (i dati salvati restano).

Per pubblicare una nuova versione: inserisci l'admin token e trascina il file APK qui sotto. Il file viene caricato direttamente su Azure Blob Storage e diventa subito scaricabile dai tablet.
