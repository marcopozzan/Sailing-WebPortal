    ⚙ Configurazione admin

    1. Autenticazione admin

    Per caricare polari/waypoint serve l'Admin Token (lo trovi su Azure: App Service → Configuration → ADMIN_TOKEN). Viene salvato solo nel browser.

    2. Configurazione Blob Storage

    Account Azure dove sono salvati polare, waypoint e tracce. I valori di default sono presi dal backend; modificali solo se ti colleghi a uno storage diverso (es. ambiente di test). Le modifiche vengono salvate solo in questo browser, non sul server.
    Importante: per cambiare l'AccountKey va modificata la connection string come Application Setting su Azure App Service (non e' configurabile da qui per motivi di sicurezza).

    --

    3. Event Hub (live stream)

    Ogni barca pubblica i dati live su un Event Hub dedicato. Configura qui quale EH usare per la barca selezionata in alto. Se EventHubName non è impostato il live per quella barca è disabilitato (l'endpoint /live/stream ritorna 503).

    --

    Carica su blob: seleziona un file .json e clicca "Carica su blob". Il file viene caricato sull'Azure Blob Storage e diventa subito disponibile al tablet.
    Scarica da blob: recupera la versione attualmente sul blob storage e te la salva sul PC. Utile per fare backup o modifiche locali.
    URL nel tablet: sul tablet, apri il browser e incolla l'URL. Il file viene scaricato. Sposta il file nella cartella Android/data/it.regolofarm.soar/files/. Riapri l'app.
    Dopo aver caricato, le viste pubbliche (Polare / Waypoints) si aggiornano automaticamente.

    Convertitore coordinate

    Il file waypoints.json usa il formato gradi-minuti (45°46.154'N). Se hai coordinate in formato decimale (es. da Google Maps), convertile qui.
